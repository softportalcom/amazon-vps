СКАЧАТЬ

Zortam Mp3 Media Studio - функциональный набор инструментов для работы с MP3 файлами, которая предоставляет широкие возможности для работы с ID3-тегами, а также позволяет извлекать аудиодорожки с CD дисков, осуществлять запись музыки на CD диски и конвертировать WAV в MP3 и обратно.

Ключевые особенности программы:

  • Автоматическое заполнение тегов, загруженных из Интернет базы Zortam Music.
  • Встроенный редактор MP3-тэгов ID3v1 и ID3v2.
  • Наличие медиапроигрывателя.
  • Встроенный MP3-органайзер.
  • Поиск песен в базе данных.
  • Возможность нормализации аудио.
  • Извлечение аудиодорожек с CD дисков с последующим сохранением в формате MP3 и WAV.
  • Пакетная конвертация файлов WAV в MP3 и наоборот.
  • Запись аудио CD.
  • Менеджер плэй-листов.
  • Поиск дубликатов MP3 файлов.
  • Одновременное переименование большого количества файлов с использованием информации из ID3 тегов.
  • Экспорт тэгов в HTML файл.
