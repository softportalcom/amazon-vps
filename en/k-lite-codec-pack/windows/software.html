K-Lite Codec Pack is a universal set of codecs and decoders for playing any multimedia files of a wide variety of audio and video formats. The package contains both 32-bit and 64-bit codecs, so it can be used equally efficiently in versions of Windows x86 and x64 operating systems. The list of supported formats is simply huge, we will note only the most popular: AVI, MKV, MP4, FLV, MPEG, MOV, TS, M2TS, MP3, OGM, WMV, 3GP, WEBM, FLAC, etc.

The codec package is available in 4 main editions, distinguished by the components included in them:

  • K-Lite Codec Pack Basic
  • K-Lite Codec Pack Standard
  • K-Lite Codec Pack Full
  • K-Lite Codec Pack Mega

The K-Lite Codec Pack Basic contains all the main codecs for playing a media resource on a PC. This is the lightest and simplest package with which you will be able to watch / listen to music and movies when using any player that is convenient for you. There are DirectShow audio/video decoding filters (LAV Audio and Video), DirectShow source and subtitle filters (LAV Splitter and VSFilter), the Codec Tweak Tool utility, as well as Icaros Thumbnail Provider and Icaros PropertyHandler extensions.

K-Lite Codec Pack Standard, in addition to all the necessary codecs, as in K-Lite Codec Pack Basic, contains additional useful utilities: a Media Player Classic Home Cinema (MPC-HC) media player specially adapted for codecs, a visualizer with advanced madVR capabilities and a detailed information about the selected MediaInfo Lite file.

The K-Lite Codec Pack Full, in addition to the content of the K-Lite Codec Pack Standart, includes some tools for processing and improving the audio / video stream: DC-Bass Source Mod, ffdshow processor and decoder, Haali Media Splitter audio/video signal separator, as well as a utility for analyzing audio and video files GraphStudioNext.

The most complete and voluminous assembly of the K-Lite Codec Pack Mega contains in addition to the K-Lite Codec Pack Full such "bonuses" as the ACM and VFW codecs, the AC3Filter filter, the Haali Matroska Muxer multiplexer and other tools.

Each user determines the choice of the version for himself, depending on the tasks set: if you just want to watch movies and listen to any music, then use the basic equipment, and if there is a need for pre - processing and editing the audio/video signal, it is better to choose something with additional functionality.

It is convenient that all updates to any of the tools, codecs or additional utilities in the K-Lite Codec Pack packages are promptly added to the latest build, so just do not forget to update your version of the collection to accommodate the most up-to-date components in the system.
