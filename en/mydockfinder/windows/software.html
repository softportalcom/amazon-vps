MyDockFinder is a comprehensive Windows customization utility from Chinese developers that makes it as similar to macOS as possible by adding missing modules from the Apple operating system, like a docking pad or Launchpad menu.

Using the application is very simple: unzip the downloaded archive and transfer it to your system drive. Before launching, if necessary, install the Microsoft Visual C++ library package, which is also present in the archive. Then launch Mydock, and you will immediately have recognizable macOS elements such as the dock bar, and menu bar as well as all icons and indicators.

In the options section, you can adjust the size of the interface elements, choose a design theme, color accents, and the ability to automatically launch the utility at the system startup. Also, you can change the size of the Dock Panel, enable or disable icon enlargement on hovering, set actions when hovering the cursor in the selected area of the desktop, and customize the behavior of windows and applications in the Dock Panel.

In the Audio settings block, you can change the standard graphics for adjusting the computer's volume level. Also here you can assign volume control keys when there are no multimedia keys on the keyboard. Besides, there is a Launchpad, where you can manually move the shortcuts of the applications you need.

MyDockFinder can be uninstalled quickly and easily. You just need to disable the utility from the context menu and delete the folder containing all the files.

Key features of MyDockFinder:

  • A handy menu bar that contains all the utility settings and some parameters of the applications you run;
  • pop-up menus for different system indicators like Wi-Fi, Bluetooth, volume indicator, etc;
  • Replacement of some system menus and dialog boxes, which begin to look more aesthetic and do not go out of style;
  • Adjusting the size of the interface elements;
  • choice of design theme and color accent;
  • Possibility to automatically start the utility at system startup;
  • adding system indicators to the menu bar;
  • ability to change the size of the Dockbar;
  • Enable/disable zooming icons when hovering over;
  • setting actions when hovering over the selected area of the desktop;
  • Adjusting how windows and applications behave in the Dockbar;
  • Displaying an indicator of CPU, RAM, video adapter load, and free space on the storage device;
  • searching and installation of appropriate wallpapers;
  • and much more...
