Defender Control is a small utility that helps you disable, enable and run Windows Defender in one click.

Entry-level Windows Defender is integrated into the operating system and runs in the background. It does not have the option to disable it for an arbitrary period of time, although such a need may well arise in some situations. For example, a more powerful antivirus, in the presence of which there is no need to run two similar applications and additional load on the processor.

Also, some applications may ask you to disable Windows Defender during installation. Disabling it will also help with resource shortages on weak computers. This is exactly the kind of situation for which the proposed program comes in handy.

The application interface consists of a couple of basic buttons, one of which disables Windows Defender in one click, the other launches it.

You can also easily go to the Windows Defender settings and use the program from the command line.
