СКАЧАТЬ

BlueTooth Driver Installer - небольшая бесплатная утилита, представляющая собой универсальный инсталлятор для BlueTooth-устройств. При запуске программа сканирует неопознанные BlueTooth-адаптеры, получает информацию о них в виде аппаратного ID и пытается модифицировать стандартные драйверы (файл WinDir\INF\Bth.inf) для установки адаптера и запуска соответствующих системных служб. Перед внесением изменений BlueTooth Driver Installer автоматически создает точки восстановления.

Полный перечень поддерживаемых BlueTooth-устройств доступен на официальном сайте.
