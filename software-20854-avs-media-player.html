СКАЧАТЬ

AVS Media Player - полнофункциональный программный медиаплеер, имеющий все функции и элементы управления обычного аппаратного плеера. Универсальный инструмент для просмотра видео, прослушивания аудио файлов и просмотра фотографий. Простой дружественный интерфейс дает возможность начать воспроизведение разных типов медиафайлов без дополнительных программ и специальных знаний.

Возможности:

  • Воспроизведение видео наиболее широко используемых форматов файлов: .3g2, .3gp, .3gp2, .3gpp, .amv, .asf, .asx, .avi, .dat, .div, .divx, .dv, .dvr-ms, .f4p, .ts, .f4v, .flc, .rm, .flv, .gvi, .ifv, .ivr, .m1v, .m2t, .m2ts, .m2v, .m4v, .mkv, .mod, .mov, .mp4, .mpe, .mpeg, .mpg, .ms-dvr, .mts, .mtv, .cdg, .nsv, .fli, .mvv, .ogg, .ogm, .ogv, .qt, .ram, .rmvb, .rv, .tp, .trp, .wmv;
  • Прослушивание аудиофайлов и аудио CD с любых цифровых носителей. Поддерживает аудио форматы: .wav, .mp3, .mpeg3, .cda, .mp2, .m4a, .wma, .ogg, .aac, .mpc, .mpp, .mp+, .aif, .aiff, .aifc, .au, .snd, .amr, .awb, .flac, .ac3;
  • При воспроизведении DVD дисков доступны все стандартные функции бытового DVD плеера;
  • Просмотр изображений и фотографий, слайдшоу. Поддерживает форматы: .bmp, .gif, .jpg, .jpeg, .jpe, .jfif, .png, .tif, .tiff, .pcx, .ras, .psd, .tga, .wmf, .emf;
  • Функция преобразования стерео сигнала в многоканальный (3.1, 5.1, 7.1);
  • Организация мультимедиа-файлов в списки воспроизведения, сохранение их в форматы M3U или PLS, загрузка ранее созданных списков воспроизведения;
  • Регулировка основных системных настроек: изменение соотношений сторон, установка количества и конфигураций колонок;
  • Поддержка субтитров для видео, изменение их цвета, типа шрифта и размера;
  • Захват снимка видео и сохранение его в один из поддерживаемых форматов графических файлов.

"AllowDVD"=dword:00000001

Короче разобрался: возможность воспроизведения DVD в сабже есть, но, по непонятным причинам, она отрублена разработчиками! Для того, чтобы включить эту возможность, нужно в редакторе реестра открыть ключ HKCU\Software\AVS4YOU\MediaPlayer\Settings и выставить параметр "AllowDVD" в 1-цу. Тогда плеер действительно будет воспроизводить DVD.
