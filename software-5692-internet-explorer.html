СКАЧАТЬ

Internet Explorer for Mac - всем хорошо известный браузер Internet Explorer от дяди Билла для Mac OS. Особенно пригодится тем, кто никак не привыкнет ко встроенному браузеру по умолчанию в Apple Mac OS - Safari.
Internet Explorer for Mac обладает всеми функциями обычного браузера под Windows, только имеет гораздо более приятный интерфейс :)
 

На данный момент этот проект более не поддерживается Microsoft.
