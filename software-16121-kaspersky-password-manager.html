СКАЧАТЬ

Kaspersky Password Manager - очень полезный инструмент для пользователей сети Интернет, который полностью автоматизирует ввод паролей и других данных на веб-страницах, тем самым делая работу во Всемирной паутине удобнее и безопаснее. При помощи встроенного генератора паролей, программа создает очень надежные пароли, а также препятствует краже пользовательских данных, помещая всю информацию в специальную базу данных на Вашем компьютере в зашифрованном виде.

Основные возможности Kaspersky Password Manager:

  • Автоматическая авторизация (Kaspersky Password Manager запоминает параметры доступа к веб-ресурсам, и при следующем посещении сайта программа автоматически подставит идентификационные данные);
  • Работа с приложениями, требующими авторизации (Outlook и др.);
  • Защита пользовательских паролей;
  • Выбор способа доступа к базе паролей (мастер-паролей, различные USB- и Bluetooth-устройства, например мобильный телефон);
  • Создание паролей при помощи встроенного генератора паролей;
  • Импорт паролей из других программ;
  • Доступ к паролям с помощью переносной версии программы;
  • Автоматическое заполнение длинных веб-форм;
  • Хранение нескольких учетных записей для одного веб-ресурса;
  • Автоматическое резервное копирование;
  • Защита от кейлоггеров;
  • Противодействие фишинг-атакам.
  • Kaspersky Password Manager для Android
Что нового в Kaspersky Password Manager 24.0.0.427?

Новые возможности:

  • Добавлена возможность удалять аккаунты из хранилища в разделе Проверка паролей.
  • Улучшения
  • Максимально возможный срок действия банковской карты, который можно указать в приложении, увеличен до 10 лет.
  • Добавлена поддержка подстановки и чтения данных из полей веб-формы, расположенных в Shadow DOM.

Исправления:

  • Исправлены проблемы с возвратом из полноэкранного режима. В полноэкранном режиме вы можете вернуть размер окна к прежнему состоянию, потянув за шапку приложения.
  • Исправлены ошибки выбора нескольких записей при удерживании клавиши Shift.
  • Исправлены проблемы с масштабированием окна приложения при использовании нескольких мониторов.
  • Также другие мелкие исправления и улучшения, связанные со стабильностью, производительностью и пользовательским интерфейсом приложения.
