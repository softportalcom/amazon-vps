СКАЧАТЬ

MSI Afterburner - функциональная утилита от компании MSI, которая предназначена для разгона видеокарт NVIDIA и AMD. С помощью программы можно регулировать напряжение питания GPU и видеопамяти, частоту ядра, управлять скоростью вращения вентиляторов, а также в режиме реального времени мониторить состояние графической карты, как например ее текущую температуру, тактовую частоту, напряжение и пр.

Пподдерживает работу с видеокартами NVIDIA GeForce 8X00 и новее, а также AMD Radeon 3000 и новее. Кроме этого пользователь может  протестировать свою видеокарту на стабильность с помощью инструмента MSI Combustor, основанного на движке известного бенчмарка FurMark.

Основные возможности MSI Afterburner:

  • Регулировка напряжения питания видеокарт и чипа видеопамяти.
  • Мониторинг переключений между режимами 2D и 3D.
  • Возможность управления системой охлаждения видеокарты.
  • Возможность создания персональных профилей (до 5 штук) с переключением по нажатию "горячей клавиши".
  • Мониторинг FPS в 3D приложениях и вывод показателей в наложенные экранные меню и системный трей.
  • Полная поддержка 64-битных приложений.
  • Возможность изменения тем оформления.
  • Возможность записать лучшие игровые моменты и поделиться ими с друзьями.
Что нового в MSI Afterburner 4.6.5?
  • Добавлена поддержка видеокарт серии NVIDIA GeForce RTX 40x0.
  • Добавлена поддержка управления напряжением для версий NVIDIA GeForce RTX 3060 Ti на базе GA103 и GDDR6x.
  • Добавлена поддержка видеокарт серии AMD RADEON RX 7900.
  • Добавлена поддержка мониторинга общей мощности платы для видеокарт AMD RADEON RX серии 7900.
  • Добавлены некоторые будущие идентификаторы AMD и NVIDIA GPU PCI DeviceID в базу данных оборудования.
  • В модуль мониторинга оборудования добавлена поддержка графических процессоров Intel Arc. Обратите внимание, что разгон и настройка графических процессоров Intel Arc в настоящее время не поддерживаются из-за ограничения поддержки API аппаратного управления Intel только приложениями x64.
  • Добавлена экспериментальная поддержка процессоров Intel 13-го поколения.
  • Добавлена экспериментальная поддержка процессоров AMD Ryzen 7xxx.
  • Источники данных об использовании ЦП в модуле мониторинга оборудования были переключены на альтернативную реализацию на основе NtQuerySystemInformation(SystemProcessorIdleInformation), поскольку традиционная устаревшая отчетность о времени простоя в NtQuerySystemInformation(SystemProcessorPerformanceInformation) не работает в текущих сборках Windows 11 22H2.
  • Добавлен обходной путь для неработающего API программирования фиксированной скорости вращения вентилятора (резервный путь, совместимый с Overdrive 5) для старых графических процессоров Overdrive 7 GCN на 22.5.2 и более новых драйверах AMD.
  • Добавлен переключатель файла конфигурации для отключения встроенного API управления напряжением надежности на видеокартах серии NVIDIA GeForce GTX 9x0 и принудительного использования устаревшего API управления напряжением P-state 2.0 на таком оборудовании. Опытные пользователи могут использовать этот переключатель для обхода блокировки управления напряжением на видеокартах серии NVIDIA Maxwell в выпусках 515 и более новых семействах драйверов.
  • Усовершенствованный анализатор формул коррекции с преобразованием формата данных, округлением и поддержкой минимальных/максимальных функций
  • Добавлен мониторинг тактовой частоты шины на основе OCMailbox для Skylake и более новых процессоров Intel. В отличие от традиционных оценок тактовой частоты шины на основе устаревших меток времени, OCMailbox поддерживает мониторинг разогнанного BCLK. Обратите внимание, что доступ к OCMailbox заблокирован конструкцией ОС, когда включен HVCI.
  • Улучшен плагин мониторинга SMART.dll. Добавлена поддержка мониторинга температуры для устройств NVMe, включая температуру вторичного контроллера для некоторых дисков Samsung NVMe.
  • Ограничение тактовой частоты по умолчанию в окне редактора кривой напряжения/частоты увеличено до 3,5 ГГц. Обратите внимание, что при необходимости вы все равно можете настроить лимиты через конфигурационный файл.
  • Расположение сервера обновлений изменено на новый URL внутри системы проверки обновлений. Старый сервер обновлений достиг EOL
